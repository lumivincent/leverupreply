LeverUp Reply Agent — Claude Code 实施手册

前置准备清单

在开始之前，确认以下环境：

# 确认 Node.js
node --version    # 需要 >= 18

# 确认 Claude Code
claude --version  # 需要已安装

# 确认 Python

# 确认 Git
git --version

准备好以下文件（从我们的对话中下载）：

  • LeverUp_Reply_Agent_Universal_Playbook.md
  • LeverUp_Reply_Agent_Claude_Code_Architecture.md

准备好 Twitter API 密钥（4 个值）：

  • TWITTER_API_KEY
  • TWITTER_API_SECRET
  • TWITTER_ACCESS_TOKEN
  • TWITTER_ACCESS_SECRET

10.3 可用回应框架

  • 承认不确定："good question, not sure I have the full picture. but from what I've seen on the docs..."

中度（growth / official 可用）：

  • 用机制事实回应："LP-free ≠ no counterparty. the risk engine handles position netting. different from the LP pool model, not the absence of risk management"
  • 用数据回应："the OI cap issue is real for LP-based models. here's how uncapped OI changes the dynamic → [具体解释]"
